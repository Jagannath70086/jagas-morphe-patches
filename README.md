## 📥 Installation

Add this repository as a patch source in Morphe Manager:

```text
https://github.com/jagannath70086/jagas-morphe-patches
```

Or use the deeplink:

```text
https://morphe.software/add-source?github=jagannath70086/jagas-morphe-patches
```

Once added, available patches will appear automatically inside Morphe Manager.

---

## 🛠️ Building From Source

Build the patch bundle locally:

```bash
./gradlew buildAndroid
```

The generated `.mpp` bundle will be located at:

```text
patches/build/libs/
```
